# Buy This Product

An online shopping platform with user authentication and product showcase.

## Features

- User registration and login
- Product showcase
- Responsive design
- Client-side data storage for GitHub Pages deployment

## Deployment

This project is deployed on GitHub Pages: https://anandofficialgit.github.io/buy-this-product/

## Local Development

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Start mock backend: `node mock-backend.cjs`

## GitHub Pages

The project uses a client-side API for GitHub Pages deployment, storing data in localStorage.
